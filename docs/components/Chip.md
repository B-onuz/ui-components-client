# Chip (Component)

Componente chip usado para listar status.

## Como utilizar

```js
import Chip from 'ui-components/lib/components/Chip'
```

```jsx
<Chip color={'primary'} text={'status'} />
```

## Variações

### Variant

```jsx
<Chip color={'secondary'} />
```

Escala no tema: `theme.variantScale`
