# Spinner (Component)

Componente para tratar estado de loading.

## Como utilizar

```js
import Spinner from 'ui-components/lib/components/Spinner'
```

```jsx
<Spinner />
```

## Variações

### Variant

```jsx
<Spinner color={'primary'} />
```

Escala no tema: `theme.colors`
